## 二、逐机制深度分析

### 机制1: 收集/图鉴系统 — V1 立即实施

**GP 分析**:
- 完形心理: 孩子看到空缺="必须填满"的冲动
- 我们的字族天然适配: 每组字族=一套图鉴
- 进度可视化=持续激励

**AD 分析**:
- 风险: 稀有度可能造成"抽卡"心态
- 缓解: 稀有度由学习进度决定(非随机)，学完一定数量自然解锁
- 稀有字应该是"奖励"不是"抽奖"

**V1 实现方案**:
```
已有基础: CollectionWall 字族收集墙
需增强:
  1. 每字族显示"未收集"灰色字卡(完形心理触发)
  2. 集齐一组→ 播放庆祝动画 + 获得勋章
  3. 进度文案: "火字旁家族: 烧、烤、炒、炸... 还差1个!"
```

### 机制2: 养成/喂养系统 — V1 实施

**GP 分析**:
- 儿童对宠物的责任感 = 最强的内在动机
- "精灵饿了" > "你今天还没学字"(家长说)
- 不同字族养出不同精灵 = 鼓励探索多字族

**AD 分析**:
- 风险: "不忍心精灵饿着" → 绕过休息限制
- 缓解: 精灵也需要睡觉! 休息时间精灵也在睡觉
- 精灵饥饿度每日上限 = 每日学习字数上限

**V1 实现方案**:
```
新增: SimplePet 组件
  - 主页顶部显示汉字精灵
  - 状态: 😊饱饱 / 😐有点饿 / 😢好饿
  - 学1字 → 喂1口 → 精灵开心
  - 每日首次学习 → 精灵说"早上好!"
  - 休息时间 → 精灵打呼噜 😴
```

### 机制3: 对战/比赛 — V2 延迟

**GP 分析**:
- 竞争适合6-9岁(规则意识形成期)
- "和昨天的自己比"比"和陌生人比"更健康
- 字族大战创意很好: "水"字族 vs "火"字族

**AD 分析** ⚠️:
- 🔴 实时对战 = 最高成瘾风险
- 🔴 社交比较可能导致挫败感(输的孩子不想玩了)
- 必须家长授权 + 限制时长
- 6岁以下禁用对战功能

**V2 方案**:
- 仅开放"和昨天的自己比"(单人模式)
- 双人对战需家长PIN确认
- 每日对战上限3局

### 机制4: 建造/创造 — V2 最大差异化

**GP 分析**:
- "学会这个字, 我的世界会新增什么?" = 终极内在动机
- 创造型玩法成瘾风险最低(无竞争/无赌博)
- 所有竞品均无此功能 → 最大差异化

**AD 分析**:
- 🟢 最低成瘾风险 — 创造是健康的
- 孩子会花时间建造(非机械重复学习)
- 家长观感极好: "孩子在学习用汉字创造世界"

**V2 方案**:
- 每学1字 → 获得1个积木
- 积木类型=字族: 木=木头块, 水=蓝色块
- 用积木搭建场景: 小木屋/花园/城堡

### 机制5: 剧情/冒险 — V2 渐进

**GP 分析**:
- Boss战设计巧妙: 用正确的字灭火/开闸/修桥
- 识字直接=战斗力, 学习即游戏
- 章节悬念驱动持续打开

**AD 分析**:
- 🟢 低风险 — 剧情有终点, 不会无限循环
- 但章节解锁速度需控制(一天最多1章)

**V2 方案**:
- 3章=3个字族(火/水/木)
- 每章5个必学字 → Boss战 → 解锁下一章

## 三、成瘾防护设计 (AD 强制要求)

| 防护 | 机制 | 触发 |
|------|------|------|
| 每日上限 | 精灵饥饿度=每日学习上限, 学满后精灵睡觉 | 不可绕过 |
| 强制休息 | 15分钟弹出护眼动画, 精灵也休息 | 不可跳过 |
| 无随机奖励 | 稀有度由进度决定, 非随机抽取 | 防止赌徒心理 |
| 家长控制 | 所有社交/对战功能需PIN确认 | 家长信任 |
| 无付费抽卡 | 付费=解锁内容, 不=抽奖 | 合规 |
| 夜间模式 | 晚8点后精灵说"该睡觉了", 降低亮度 | 保护睡眠 |
