# Research Organization

## 6 Research Roles

| Role | Focus | Method |
|------|-------|--------|
| **CD** 儿童发展专家 | Cognitive stages, game psychology, age-appropriate design | Development theory + game mechanics analysis |
| **MA** 市场分析师 | Market size, pricing, trends, TAM/SAM/SOM | Industry reports + competitor financials |
| **CI** 竞品情报 | Feature tracking, user reviews, version monitoring | App Store scraping + deep product testing |
| **UR** 用户研究员 | Parent interviews, child observation, diary studies | Interview protocols + observation guides |
| **DA** 数据分析师 | User behavior, retention funnel, conversion | Event tracking design + cohort analysis |
| **CM** 社群调研 | Parent communities, KOL content, word-of-mouth | Social listening + sentiment analysis |
